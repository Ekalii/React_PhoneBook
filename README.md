## create-react-app installization
* Open Git Bash
* `npm install -g create-react-app` 또는</br>
  `yarn global add create-react-app` 을 통해 설치
* git clone
* `cd PROJECTNAME` 후 `npm install`
* 10분정도 기다린다!!(싸지방ㅠㅠㅠㅠㅠㅠㅠ)
* `yarn start`
* 끝!
